Maak 1 object:

In object Earthquake:
-In de object opties
Gewoon de normale settings behalve dat je 'visable' uit moet zetten

-In de create event
--> viewsidesmax = 32 // hoe hoger dit getal, hoe langer en harder de aardbeving

-In de step-event
--> viewsidesmax -= 1 // dit regelt het minder snel gaan van de schudden. Hoe hoger dit getal, hoe korter de aarbeving.
--> if (viewsidesmax < 1) {viewsidesmax = }
--> view_right += random(viewsidesmax)-random(viewsidesmax) // dit regelt het schudden van de view van links en rechts
view_bottom += random(viewsidesmax)-random(viewsidesmax) // dit regelt het schudden van de view van top en bodem

Opmerkingen
--> Gebruik een goed zichtbare achtergrond met meerdere kleuren en/of vormen.
--> In de room waar je de Earthquake gebruikt moet je 'Enable use of views' en
'visable when room starts' in de view tab aanvinken.
--> Als je van plan bent geen ViewBorder gebruikt, vink dan'Draw background color' aan in de background tab. Deze kleur moet ongeveer gelijk zijn aan die van het achtergrondplaatje.
